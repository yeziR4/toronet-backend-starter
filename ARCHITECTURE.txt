================================================================================
  File Layout
================================================================================

  src/
  ├── index.ts              Entry point, Express server bootstrap
  ├── config/
  │   └── env.ts            Zod-validated environment variables
  ├── sdk/
  │   ├── client.ts         Toronet SDK init & config access
  │   ├── wallet.ts         Create / import / verify wallets
  │   ├── blockchain.ts     Query chain status, blocks, transactions
  │   ├── balance.ts        TORO, fiat, and token balances
  │   ├── tns.ts            TNS name registration & resolution
  │   ├── kyc.ts            KYC setup, verification, enrollment
  │   ├── currency.ts       Transfer, inter-wallet, exchange rates
  │   ├── bridge.ts         Multi-chain bridge fees & transfers
  │   ├── products.ts       Product CRUD
  │   └── deployer.ts       Smart contract deployment & registration
  ├── routes/
  │   ├── index.ts          Route aggregator
  │   ├── wallet.routes.ts
  │   ├── blockchain.routes.ts
  │   ├── balance.routes.ts
  │   ├── tns.routes.ts
  │   ├── kyc.routes.ts
  │   ├── currency.routes.ts
  │   ├── bridge.routes.ts
  │   ├── products.routes.ts
  │   └── deployer.routes.ts
  ├── middleware/
  │   └── error-handler.ts  Centralized error → JSON response
  ├── types/
  │   ├── api.ts            ApiResponse / ApiError interfaces
  │   └── errors.ts         ToronetError, NotFoundError, SdkError
  └── utils/
      ├── logger.ts         Pino logger instance
      └── response.ts       sendSuccess / sendError helpers

  tests/
  ├── wallet.test.ts
  └── sdk.test.ts

================================================================================
  Request / Response Flow
================================================================================

  Client Request  ──►  Express Route  ──►  SDK Module  ──►  torosdk  ──►  Toronet
       │                    │                    │                │              │
       │                    │                    │                │              │
       │                    │              SdkError thrown      │         Error thrown
       │                    │                    │                │              │
       │              errorHandler catches      │                │              │
       │              ──► JSON error response  ◄────────────────┘              │
       │                                                                        │
       ◄───── JSON success response (ApiResponse<T>) ◄──────────────────────────┘

  Success shape:  { success: true,  data: <T> }
  Error shape:    { success: false, error: { message, code?, statusCode } }
